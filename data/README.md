# data/

`rockyou.txt` is not committed to this repository. Download it separately and place it at `data/rockyou.txt` before running the preprocessing pipeline. It is widely available online and used in most password analysis work.

The following two files are generated by `src/python/frequency_analysis.py` and should not be edited by hand:

`candidates_ranked.txt`: the frequency-ranked candidate list produced from the RockYou wordlist. One candidate per line, ordered from most to least statistically likely. This file is the input to the C++ cracker binary.

`substitution_rules.json`: the computed substitution pattern frequencies extracted during analysis (e.g., `a -> @`, `e -> 3`). Used internally by the frequency analysis pipeline and documented in the project write-up.
